# Yuvraj Pharande — Portfolio

Source for my personal portfolio site, live at **[yuvrajcoep.github.io](https://yuvrajcoep.github.io/)**.

A single-page, fully responsive developer portfolio built with **plain HTML & CSS** (no framework, no build step) and a touch of vanilla JavaScript for the mobile menu and scroll animations. Hosted on GitHub Pages.

## Sections
- **Hero** — intro and headline
- **About** — background and focus
- **Skills** — languages, backend, frontend, AI/ML, and tooling
- **Projects** — featured work ([AI Business Accelerators](https://github.com/Yuvrajcoep/ai-business-accelerators), [Email Triage Environment](https://github.com/Yuvrajcoep/email-triage-env))
- **Contact** — LinkedIn

## Run locally
It's static — just open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 8000
# then visit http://localhost:8000
```
